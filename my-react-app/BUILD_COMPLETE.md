## 🚀 Quick Start (3 Steps)

### Step 1: Install Dependencies
```bash
cd my-react-app
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
Navigate to `http://localhost:5173`

   - Change name, title, and description

2. **Update Your Skills**
   - File: `src/Project/Components/Skills.jsx`
   - Edit skill categories and proficiency levels (0-100)

3. **Add Your Projects**
   - File: `src/Project/Components/Projects.jsx`
   - Add 6 of your best projects with descriptions and links

4. **Update Contact Info**
   - File: `src/Project/Components/Contact.jsx`
   - Phone, email, location

5. **Update Social Links & Footer**
   - File: `src/Project/Components/Footer.jsx`
   - Add your GitHub, LinkedIn, etc.

--secondary-color: #a29bfe;    /* Light shade */
--accent-color: #00b894;       /* Highlight */
